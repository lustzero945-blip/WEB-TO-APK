Informations & Documentation Technique

Lust Web APK Builder

Version 3.0.0 (Progressive Web App)

Cette application est un studio complet de configuration et de simulation de compilation Web APK, entièrement réécrite en PWA moderne pour permettre un fonctionnement fluide, installable et rapide sur tous vos supports mobiles ou navigateurs Web sans l'encombrement d'Android Studio.

Avantages de la PWA

  • **Zéro installation requise** pour commencer, utilisable depuis n'importe quel navigateur.
  • **Fonctionnement 100% Hors ligne** via cache statique et Service Workers intégrés.
  • **Déploiement direct sur Netlify** en une seule commande ou liaison Git.

Intégration d'IA Gemini

  • Analyse sémantique intelligente basée sur le modèle `gemini-3.5-flash`.
  • **Sécurité renforcée** : Les clés privées sont gérées par variables d'environnement côté serveur, évitant toute fuite côté client.
